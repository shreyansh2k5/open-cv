# OpenCV Projects — Step by Step

## Setup
```bash
pip install opencv-python mediapipe numpy
```

## Run in order
| File | Topic |
|------|-------|
| 01_basics.py | Images, colors, drawing |
| 02_processing.py | Blur, edges, threshold |
| 03_color_detection.py | HSV masking, contours |
| 04_webcam.py | Live webcam + FPS counter |
| 05_hand_gestures.py | Hand landmarks + finger counter |

Press **Q** to quit any webcam window.
